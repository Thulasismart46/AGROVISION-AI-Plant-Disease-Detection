# 🌱 AGROVISION

AI-Based Plant Disease Detection System

## 📖 About

AGROVISION is a Machine Learning-powered web application that detects plant diseases from leaf images. Farmers and agricultural professionals can upload plant leaf images and receive disease predictions instantly.

## 🚀 Features

- Plant disease detection using AI/ML
- Image upload functionality
- Real-time disease prediction
- User-friendly web interface
- Responsive design

## 🛠️ Technologies Used

- Python
- HTML5
- CSS3
- JavaScript
- Machine Learning
- Flask
